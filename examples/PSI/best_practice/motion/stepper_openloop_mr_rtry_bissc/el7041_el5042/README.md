# EL7041-0052 and EL5042 Startup Example

This directory contains the runnable startup example for the open-loop stepper
with motor-record retries described in the parent directory.

- Parent overview: [../README.md](../README.md)
- Startup file: `startup.cmd`
- Configuration files: `cfg/axis.yaml`, `cfg/enc_open_loop.yaml`

Run it from this directory:

```sh
iocsh.bash startup.cmd
```
